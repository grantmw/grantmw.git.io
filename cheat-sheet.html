What is SQL?

SQL stands for Structured Query Language. It is a language to communicate with the database management system (DBMS). There are slightly different versions for different vendors or databases.

SQL can interact with databases in a number of ways. It can:

  • Query a database.
  • Retrieve data from a database.
  • Update, insert, delete or change records in a database.
  • Create new databases or delete old ones.
  • Set permissions within a database to determine who can read/write/access its data.

Conventions in SQL

There are a number of conventions in SQL that are standard for the language.

  • Table names should be pluralized.
  • Every command needs to end with a semicolon for it to run.
  • Commands are not case sensitive, but convention states that basic commands are written in all caps, i.e. SELECT or FROM.
  • The spaces in the names of rows and columns should be replaced by underscores.

Cardinality

In SQL, cardinality refers to the uniqueness of data in a database table. With each data set being a column of data, the cardinality refers to the uniqueness of the data per instance. A data point (column) with a high cardinality will have different value for every instance (row), where a data point with a low cardinality will have the same value over many instances. The cardinality can help with structure and readability.

The cardinality of a table can be expressed as a number - if you are assigning values from a list of directions (north, south, east, west), this column would have a cardinality of 4, because there are four possible values.

Additionally, cardinality can represent the relationship of one table with another. The one-to-one, one-to-many, and many-to-many are common cardinalities.

How to select and order values from SQL databases

  • To display all data in a table:

    SELECT * FROM table_name;

  • To select specific columns in a table:

    SELECT name_column, name_column2
    FROM table_name;

  • To order values by ascending or descending:

    ORDER BY column_name DESC
    or
    ORDER BY column_name ASC

  • To select certain instances within columns*:

    WHERE column_name='value_of_instance'

    * there can also be different operators used with the where clause, such as any numerical operator, or built-in systems specific to SQL, or <, >, etc.

Vocab and terms

Here are some commonly used terms when talking about SQL:

  • entity - objects to represent
  • attribute - something that describes an entity
  • table - represents entities with attributes
  • column - separates each attribute
  • rows - separates instances of the entity
