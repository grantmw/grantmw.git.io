Parlay - Make Bets with Friends

Although currently down for maintenance (Venmo does not agree with the sentiment of the application), Parlay is a single-page application for friends to make non-consequential, harmless bets with each other. Users are able to make bets, accept bets, and conveniently exchange money when a winner is declared. The tech-stack includes: Rails, Angular.js, and the Venmo API.

After coming up with the idea for the application, I acted as the team lead, collaborating with a few friends. What I did: Integrated Venmo OAuth for authorization and authentication; designed the relational database which allowed users to exchange real money with Venmo friends; built out routes using Angular.

  • Check out the code: Github Repo

See how it works below:

Login using Venmo OAuth

Users can propose a bet.

Users are able to accept or decline bets. Bets that have not been "shook on" live here.

Once bet receivers accept a bet, the bet is on. The winner of the bet can be declared anytime; both parties must declare the same winner.

After the winner is declared, the loser has the option to pay while the winner awaits payment.

Winners receive real payments through Venmo's API.
